## 🚀 Project Overview
**Opinor** is a comprehensive feedback and review system for businesses. It consists of a robust backend API and a high-fidelity mobile application.

### Core Components
1. **Opinor API (Backend)**: Built with NestJS and PostgreSQL. It handles authentication (JWT), user management, join requests, and business logic for feedback collection.
2. **Opinor Mobile (Frontend)**: An Expo-based React Native application focusing on a premium user experience, including data visualization and interactive feedback mechanics.
3. **Figma Assets**: Extracted UI designs used as a reference for pixel-perfect implementation.

---

## 🛠️ Technology Stack

### Backend (`opinor-api-main`)
- **Framework**: NestJS
- **Database**: PostgreSQL with TypeORM
- **Authentication**: JWT (Passport.js)
- **Validation**: Class-validator / Class-transformer
- **Key Features**: Auto-generating credentials documents (`docx`), email notifications (`nodemailer`), and comprehensive Swagger documentation.

### Frontend (`opinor-mobile`)
- **Framework**: React Native (Expo SDK 55+)
- **State Management**: Zustand
- **Data Fetching**: TanStack Query (React Query)
- **Navigation**: React Navigation (Native Stack & Bottom Tabs)
- **Styling/UI**: Reanimated, Gesture Handler, Lucide Icons, Bottom Sheet.
- **Charts**: `react-native-chart-kit` for data visualization.
